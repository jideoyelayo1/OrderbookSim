\doxysubsection{Detailed Description}
Stores details of an individual order in the orderbook. 

The \doxylink{class_order_detail}{Order\+Detail} class contains information about an order, including its type, side (buy/sell), price, quantity, and creation timestamp. It provides functions to retrieve and set these values.
