\doxysubsection{Detailed Description}
Represents a modification to an existing order. 

The \doxylink{class_order_modify}{Order\+Modify} class is used to store and apply modifications to an order. It includes details such as the modified order\textquotesingle{}s ID, price, side (buy/sell), and quantity. The class also provides a method to convert these modifications into a new \doxylink{class_order}{Order} object.
