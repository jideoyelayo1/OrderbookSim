\doxysubsection{Detailed Description}
Defines the \doxylink{class_order}{Order} class and related types for managing orders in a trading system. 

This file contains the \doxylink{class_order}{Order} class, which represents a trading order with attributes such as order type, ID, side, price, and quantity. It also provides functionality to manage and update the state of an order.
