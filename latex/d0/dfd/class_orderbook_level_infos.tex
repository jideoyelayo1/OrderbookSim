\doxysubsection{Detailed Description}
Holds the bid and ask levels for an orderbook. 

The \doxylink{class_orderbook_level_infos}{Orderbook\+Level\+Infos} class stores bid and ask levels represented by Level\+Infos objects. It provides methods to access these levels, which can be useful for orderbook analysis and data dissemination.
