\doxysubsection{Detailed Description}
Stores details of a matched order, including price, quantity, and time. 

The \doxylink{class_matched_order_detail}{Matched\+Order\+Detail} class represents an individual matched order in a trading system. It contains the order\textquotesingle{}s price, quantity, and timestamp, and provides accessors for retrieving each of these details.
