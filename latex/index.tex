\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md1}
Orderbook\+Sim is a C++ application that simulates an order book used in financial markets. It efficiently manages and matches buy and sell orders, and calculates the volume-\/weighted average price (VWAP). The project also includes a neural network built from scratch to predict sell prices, and now features GUIs for adding and viewing the order book in Python. This simulation helps users understand how order books work and how neural networks can be applied to financial data.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Running the program}{Running the program}}\label{index_autotoc_md2}

\hypertarget{index_autotoc_md4}{}\doxysubsection{\texorpdfstring{Orderbook}{Orderbook}}\label{index_autotoc_md4}
The main class representing the order book. It manages buy and sell orders, matches them based on price-\/time priority, maintains an order history, and calculates VWAP.\hypertarget{index_autotoc_md5}{}\doxysubsection{\texorpdfstring{Order}{Order}}\label{index_autotoc_md5}
Represents an individual order with attributes such as order type, side (buy/sell), price, and quantity.\hypertarget{index_autotoc_md6}{}\doxysubsection{\texorpdfstring{Order Types}{Order Types}}\label{index_autotoc_md6}
The project supports various order types including\+:
\begin{DoxyItemize}
\item {\bfseries{Good Till Cancel}}\+: The order remains active until filled, canceled, or it expires.
\item {\bfseries{Fill and Kill}}\+: The order is filled as much as possible, and any remaining quantity is canceled immediately.
\item {\bfseries{Market \doxylink{class_order}{Order}}}\+: Filled at the best available price.
\item {\bfseries{Good For Day}}\+: Similar to Good Till Cancel, but automatically canceled at the end of the trading day.
\item {\bfseries{Fill or Kill}}\+: Must be filled immediately and entirely or canceled.
\end{DoxyItemize}\hypertarget{index_autotoc_md7}{}\doxysubsection{\texorpdfstring{Order Modify}{Order Modify}}\label{index_autotoc_md7}
Handles modifications to existing orders, such as price or quantity adjustments.\hypertarget{index_autotoc_md8}{}\doxysubsection{\texorpdfstring{Trade}{Trade}}\label{index_autotoc_md8}
Represents a trade executed between a buy and sell order.\hypertarget{index_autotoc_md9}{}\doxysubsection{\texorpdfstring{VWAP (Volume-\/\+Weighted Average Price)}{VWAP (Volume-\/\+Weighted Average Price)}}\label{index_autotoc_md9}
