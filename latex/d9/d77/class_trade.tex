\doxysubsection{Detailed Description}
Represents a completed trade, containing information about the bid and ask sides. 

A \doxylink{class_trade}{Trade} object contains bid and ask details, encapsulated in \doxylink{struct_trade_info}{Trade\+Info} objects. It provides methods to access these details, facilitating the retrieval of trade information in a trading system.
