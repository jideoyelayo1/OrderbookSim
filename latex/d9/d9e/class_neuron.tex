\doxysubsection{Detailed Description}
Represents a single neuron within a neural network layer. 

The \doxylink{class_neuron}{Neuron} class handles the neuron\textquotesingle{}s output value, weights for connections to subsequent neurons, and methods for updating weights and calculating gradients.
