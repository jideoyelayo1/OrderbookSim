\doxysubsection{Detailed Description}
Represents a neural network composed of layers of neurons. 

The \doxylink{class_net}{Net} class manages the structure and operations of a neural network, including setting up the network topology, performing forward propagation, and executing backpropagation to adjust weights based on errors.
