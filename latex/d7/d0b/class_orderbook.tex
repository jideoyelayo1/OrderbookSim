\doxysubsection{Detailed Description}
Manages and matches orders in a trading environment. 

The \doxylink{class_orderbook}{Orderbook} class maintains separate collections for bids and asks, as well as provides functions for adding, canceling, modifying, and matching orders. It also includes features for managing Good For Day orders and saving the orderbook state to JSON.
