\doxysubsection{Detailed Description}
Manages a neural network\textquotesingle{}s structure, data, and training process. 

The \doxylink{class_neural_network}{Neural\+Network} class provides functionality to set up the network topology, run the network with input and target values, and retrieve the results. It acts as a high-\/level manager for the neural network operations.
